# Leaflet.geoJSON.dateLine
Leaflet plugin that plots GeoJSON point data on both sides of the International Date Line at the same time.

### Usage Example

```js
var layer = L.geoJSON.dateLine(<Object> geojson?, <Object> options?);
```
